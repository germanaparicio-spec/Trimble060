# Trimble 0-60

This project is a simple webpage dedicated to all things Trimble 0-60 on GitHub.

## Purpose

The site acts as a small landing page for the Trimble 0-60 project and links back to the GitHub repository.

For more background on the Trimble 0-60 Challenge, visitors should also check the official Trimble page:

- `https://www.trimble.com/en/our-company/events/dimensions/experiences/0-60-challenge`

Open `index.html` in a web browser to view the site locally.
